Detailed Description

Represents a relative directory path.

The extension method PathHelpers.ToRelativeDirectoryPath(string) can be called to create a new IRelativeDirectoryPath object from a string.

Member Function Documentation

◆ GetAbsolutePathFrom()

new IAbsoluteDirectoryPath LegendaryExplorerCore.Gammtek.Paths.IRelativeDirectoryPath.GetAbsolutePathFrom ( IAbsoluteDirectoryPath  pivotDirectory)

Resolve this relative directory from pivotDirectory . If this directory is "..\Dir2" and pivotDirectory is "C:\Dir1\Dir3", the returned absolute directory is "C:\Dir1\Dir2".

The returned directory nor pivotDirectory need to exist for this operation to complete properly.

Parameters
pivotDirectoryThe pivot directory from which the absolute path is computed.
Exceptions
System.ArgumentExceptionAn absolute path cannot be resolved from pivotDirectory . This can happen for example if pivotDirectory is "C:\Dir1" and this relative directory path is "..\..\Dir2".
Returns
A new absolute directory path representing this relative directory resolved from pivotDirectory .
