Detailed Description

Represents a path containing variable(s) defined with the syntax (case-insensitive) . Such path must be prefixed with a variable and can then contain more variables.

Gets all variables contained in this path, ordered from first one (the prefix variable) to the last one.

For example, for the path $(Variable2Name) this property getter returns ["Variable1Name","Variable2Name"].
