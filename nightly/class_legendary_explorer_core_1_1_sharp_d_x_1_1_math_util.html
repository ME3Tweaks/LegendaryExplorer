Checks if a and b are almost equals, taking into account the magnitude of floating point numbers (unlike WithinEpsilon method). See Remarks. See remarks.

Parameters
aThe left value to compare.
bThe right value to compare.
Returns
true if a almost equal to b, false otherwise

The code is using the technique described by Bruce Dawson in Comparing Floating point numbers 2012 edition.
