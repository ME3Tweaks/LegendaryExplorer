Detailed Description

Represents a .tlk file, as used by ME2/ME3/LE2/LE3. For ME1/LE1 TLK, use ME1TalkFile. Used for reading from .tlk files, and writing an xml representation.

Writing a .tlk file is done with the HuffmanCompression class
