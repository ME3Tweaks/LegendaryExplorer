Represents the tlk embedded in a BioTalkFile export, which is used in ME1/LE1. For ME2/ME3/LE2/LE3TLK, use ME2ME3TalkFile.

Parameters

withFileNameOptional: If true, the filename will be appended to the returned string

noQuotesOptional: If the returned string data should be in quotes or not. Setting this to false makes the string return the exact TLK string
maleOptional: Unused in Game 1 as TLKs are fully split.
