enum LegendaryExplorerCore.Gammtek.Paths.PathType

Defines a path mode, absolute, relative or prefixed with an environment variable.

Since the a PathMode value can be variable, this enumeration can favor a generic way of coding in certain situations, by replacing calls to getters like P:NDepend.Path.IPath.IsAbsolutePath, P:NDepend.Path.IPath.IsRelativePath or P:NDepend.Path.IPath.IsEnvVarPath by calls to IPath.PathType.
