Parameters
entryThe import to resolve
globalCachePackage cache that contains global files like SFXGame, Startup, etc. The cache will not be modified but can be used to reduce disk I/O.
lookupCachePackage cache if you wish to keep packages held open, for example if you're resolving many imports
localizationThree letter localization code, all upper case. Defaults to INT.
unsafeLoadIf we are only testing for existence; use unsafe partial load. DO NOT USE THE RESULTING VALUE IF YOU SET THIS TO TRUE
gameRootOverrideThe root path of the game. If null, the default path will be used
Returns
