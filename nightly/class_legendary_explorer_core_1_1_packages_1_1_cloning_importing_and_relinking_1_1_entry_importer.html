Attempts to resolve the import by looking at associated files that are loaded before this one. This method does not use a global file cache, the passed in cache may have items added to it.

Parameters
entryThe import to resolve
localCachePackage cache if you wish to keep packages held open, for example if you're resolving many imports
localizationThree letter localization code, all upper case. Defaults to INT.
Returns
The resolved export, or null if the referenced import could not be found

Parameters
