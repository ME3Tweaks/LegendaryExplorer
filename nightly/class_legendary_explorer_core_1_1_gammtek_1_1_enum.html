static TEnum LegendaryExplorerCore.Gammtek.Enum< TEnum >.ConvertFromOtherEnumValue ( object  inputEnumValue)
inlinestatic

Converts a specific enum value from one specific enum type to another enum type by it's name.

For example, to convert Catel.Services.CameraType to Microsoft.Devices.CameraType, use the following code:

ConvertEnum<Microsoft.Devices.CameraType>(Catel.Services.CameraType.Primary);

Parameters
inputEnumValueThe input enum value.
Returns
The converted enum value.
Exceptions
ArgumentNullExceptionThe inputEnumValue is null.
ArgumentExceptionThe inputEnumValue is not of type Enum.
ArgumentExceptionThe value of inputEnumValue cannot be converted to a value of TEnum .
