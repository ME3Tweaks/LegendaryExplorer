Detailed Description

Represents a relative file path.

The extension method PathHelpers.ToRelativeFilePath(string) can be called to create a new IRelativeFilePath object from a string.

Member Function Documentation

◆ GetAbsolutePathFrom()

new IAbsoluteFilePath LegendaryExplorerCore.Gammtek.Paths.IRelativeFilePath.GetAbsolutePathFrom ( IAbsoluteDirectoryPath  pivotDirectory)

Resolve this relative file from pivotDirectory . If this file is "..\Dir2\File.txt" and pivotDirectory is "C:\Dir1\Dir3", the returned absolute file is "C:\Dir1\Dir2\File.txt".

The returned file nor pivotDirectory need to exist for this operation to complete properly.

Parameters
pivotDirectoryThe pivot directory from which the absolute path is computed.
Exceptions
System.ArgumentExceptionAn absolute path cannot be resolved from pivotDirectory . This can happen for example if pivotDirectory is "C:\Dir1" and this relative file path is "..\..\Dir2\File.txt".
Returns
A new absolute file path representing this relative file resolved from pivotDirectory .
