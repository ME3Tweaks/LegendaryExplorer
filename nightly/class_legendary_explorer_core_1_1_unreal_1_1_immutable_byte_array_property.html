Detailed Description

Replacement for ArrayProperty<T> where T is ByteProperty. Arrays of bytes are almost always embedded data (such as swfs) that are thousands or millions of bytes long. Allocating a ByteProperty for each byte is grossly inefficient, so this just keeps it internally as a byte[]
