Deserializes properties from rawStream .

For advanced use. You should almost always just call ExportEntry.GetProperties

Parameters
exportThe ExportEntry these props are from
rawStreamThe stream to read the props from. This is assumed to taken from export 's data.
typeNameThe ClassName of the export, or the Name of the Struct being read.
includeNonePropertyOptional: Should the returned PropertyCollection include the ending NoneProperty
requireNoneAtEndOptional: Should this return an empty collection if there is no NoneProperty at the end
entryOptional: The object these properties are logically in. Defaults to export , but when parsing a DefaultObject or ScriptStruct defaults it should be the Class definition they belong to
packageCacheOptional: A cache that will be used for any neccesary lookups (such as default struct values)
Returns
A collection of properties read from rawStream until a NoneProperty was found, or until the end of the stream if requireNoneAtEnd is false
